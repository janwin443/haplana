# HaPlana 📚

**HaPlana** ist ein moderner, plattformübergreifender Hausaufgabenplaner, der zu **100 % auf Python** basiert. Das Projekt verfolgt einen hybriden UI-Ansatz, um sowohl auf dem Desktop als auch auf mobilen Endgeräten eine optimale Performance und Benutzerfreundlichkeit zu bieten.

---

## 🚀 Features

* **Plattformübergreifend:** Volle Unterstützung für Desktop (Windows, macOS, Linux) und Mobile (Android, iOS).
* **Intelligente Verwaltung:** Erfasse Hausaufgaben mit Fälligkeitsdatum, Fach und Priorität.
* **Status-Übersicht:** Behalte den Überblick über erledigte und anstehende Aufgaben.
* **100% Python:** Einfach zu warten, zu erweitern und zu verstehen.

## 🛠 Technische Architektur

HaPlana nutzt eine geteilte Logik-Basis, verwendet aber spezialisierte Frameworks für die grafische Oberfläche:

| Plattform | Framework | Fokus |
| :--- | :--- | :--- |
| **Desktop** | **Tkinter** | Leichtgewichtig, native Performance und geringer Ressourcenverbrauch. |
| **Mobile** | **Kivy** | Touch-optimierte Oberflächen und Multi-Touch-Unterstützung. |
